# 📻 Esma3 Radio

Modern Arabic-first radio streaming platform built with a professional fullstack architecture using Next.js, TypeScript, PostgreSQL, and AI integrations.

---

## ✨ Overview

Esma3 Radio is a modern web platform designed to deliver a smooth and immersive radio streaming experience with beautiful UI/UX, multilingual support, smart recommendations, and advanced integrations.

The project focuses on performance, scalability, responsiveness, and creating a modern Arabic-first experience.

---

## 🚀 Features

- 🎧 Live radio streaming
- 🤖 Smart assistant
- 🌍 Arabic & English support (i18n)
- 🔔 Push notifications system
- 📱 Fully responsive design
- 🔐 Authentication system
- 📊 Analytics dashboard
- ⚡ Optimized performance
- 🌙 Dark mode support
- 🔎 Search functionality
- ✨ Smooth animations and transitions

---

## 🛠 Tech Stack

### Frontend
- Next.js
- React 19
- TypeScript
- Tailwind CSS
- shadcn/ui
- Framer Motion

### Backend
- Next.js API Routes
- PostgreSQL
- Prisma ORM
- NextAuth

### Integrations
- Smart Assistant
- Web Push Notifications
- Radio APIs
